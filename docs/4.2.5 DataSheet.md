# Toolkit - Data Sheets

### [Ir a Link: DataSheet](https://javier-rocha.github.io/framework-ia-distrito.github.io/4.2.5%20DataSheet.html)

El Data Sheet es un instrumento de documentación diseñado para describir de manera exhaustiva las características, calidad, origen, composición, riesgos y limitaciones de un conjunto de datos utilizado para entrenar, validar o ejecutar un sistema de IA. Su propósito es garantizar que las entidades públicas comprendan la procedencia y las implicaciones del dataset antes de su uso.

### Alineación con Marcos Normativos

- **Circular 002 de la SIC:** El Data Sheet facilita el cumplimiento al documentar el origen, finalidad, riesgos, y medidas de seguridad de los datos, permitiendo auditorías y demostrando el cumplimiento normativo.
- **CONPES 4144:** Se alinea con los principios de gobernanza de datos, trazabilidad, identificación de sesgos, transparencia y uso responsable, configurándose como la "hoja de vida" del dataset.
